# Screenshots

Store full-size device or emulator captures here. Use lowercase names in the
form `<watch-face>-<mode>-<device>.png`, for example:

```text
radar-sweep-active-pixel-watch-3.png
artificial-horizon-ambient-emulator.png
```

Before committing a capture, crop unrelated UI where practical and remove
serial numbers, network addresses, notifications, account information, and
other personal data. Picker preview assets required by Android remain in each
project's `watchface/src/main/res/drawable/` directory.
